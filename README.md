# Appolly Landing Page

A responsive marketing landing page for **Appolly**, a mobile finance and payments app. Built with React, Vite, and Tailwind CSS from a Figma-based design.

## Running the code

```bash
npm i
npm run dev
```

Install dependencies, then start the development server.

## Primary industry: Fintech & digital payments

The UI mockups and copy position Appolly as a **personal finance and payments app** for:

- Sending and receiving money
- Tracking spending and transaction history
- Viewing dashboards and analytics
- Managing profiles and contacts from mobile

This aligns with **consumer fintech** products such as digital wallets, P2P payment apps, personal budgeting tools, and neobank companion apps.

## Secondary industries

| Industry | Fit |
|----------|-----|
| **SaaS / mobile-first startups** | Classic app-launch layout: hero, features, screenshots, download CTAs |
| **Digital banking & neobanking** | Transaction views, budgets, and spending insights |
| **Agency / product studio showcase** | Polished UI for app dev or design portfolio case studies |

The headline and structure are broad enough to adapt to other B2C mobile apps, but the current assets and content are finance-focused.

## Business models

| Model | How this page supports it |
|-------|---------------------------|
| Freemium app | Download CTAs with premium features highlighted |
| Subscription SaaS | Blog, newsletter, and feature sections for retention |
| B2C consumer app | App Store and Google Play badges |
| Startup pre-launch | Hero messaging plus newsletter signup in the footer |
| Established fintech brand | Team, testimonials, and download stats for trust |

## Target audience

- Everyday consumers who want simple money management on mobile
- Young professionals used to app-based banking and payments
- Small business owners tracking activity and payments
- Mobile-first users on iOS and Android

## Best-fit use cases

- Fintech startup launching a payments or budgeting app
- Digital wallet or P2P transfer product
- Personal finance app with dashboard and analytics
- Mobile app agency presenting a finance app concept
- Neobank or challenger bank marketing its mobile app

## Page sections

- **Hero** — Value proposition and app store download
- **About** — Product purpose and key benefits
- **Features** — Core selling points around design, platform support, and support
- **Screenshots** — App interface carousel (dashboard, expenses, stats, profile)
- **How to use** — Video walkthrough
- **Download** — Store badges and usage statistics
- **Team** — Credibility and company story
- **Testimonials** — Social proof
- **Blog** — Content marketing and SEO
- **Contact & footer** — Support details and newsletter

## Tech stack

- React 18
- Vite
- Tailwind CSS 4
- Motion (animations)
